# Car-Model-Detector
MCS 273P Final Project



#To run
1) Get a GROQ API Key
2) Create a .env file in the backend folder and put the API Key there (GROQ_API_KEY = "YOUR API KEY HERE")
3) Open up two terminals.
4) In the first one, cd into the backend folder and type "uvicorn app:app --reload"
5) In the second one, cd into the frontend folder and type "npm start"
6) When running subsequent times, just do steps 3-5
